Variables php

In php, you can change the data type of any variable whenever you want. we use the dollar sign ("$") to make variables in php.


$car= Tesla;
echo $car;


You can't use a number as the first character of a variable name.S

$one_book = name; ✔
$1_book = name;❌


PHP allows you to access a variable by using indirect reference. In direct reference is the process of generating and accessing a variable by runtime.

$car=BMW;
$$car=Nissan;
echo $car;


If you run the above code you will gonna get Nissan not BMW. There is no limit regarding the number of indirect reference.
